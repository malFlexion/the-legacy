Ask anything about Legacy

The finetuned model knows the current meta (Dimir Tempo, Sneak and Show, Oops! All Spells, etc.), card evaluations, budget substitutions, and rules. Card names in responses are resolved against Scryfall and shown below each message.

Import & analyze a decklist

Paste a decklist (plain text, one card per line) or a Moxfield / MTGGoldfish URL. Get archetype classification, card images, mana curve, and LLM-generated analysis.

Parsed decklist, mana curve, and card grid will appear here.

Goldfish the deck

Draw opening hands with London Mulligan, or run statistical simulations. The turn engine plays cantrips before threats and tracks mana efficiency, combo assembly, and cast rates.

Opening hand, turn-by-turn log, or aggregate stats will appear here.

Budget tier generator

Deterministic — no LLM involved. Takes a decklist and produces Full / Mid / Budget versions using curated substitutions (e.g., Underground Sea → Watery Grave). Prices from Scryfall, trade-offs included.

Full / Mid / Budget tiers with prices and substitutions will appear here.
